Phần II. Trắc nghiệm Đúng Sai

Câu Nội dung câu hỏi Đúng Sai
1aCác nhà thiết kế chỉ cần biết về HTML và không cần quan tâm đến CSS hay JavaScript.X
1bThiết kế trang web có thể bao gồm việc tạo ra các hình ảnh, biểu tượng, và đồ họa khác.X
1cCác trang web hiện đại thường sử dụng cả HTML, CSS và JavaScript để hoạt động hiệu quả.X
1dViệc tối ưu hóa SEO không quan trọng trong quá trình thiết kế trang web.X
2aPhần tử giả ::first-letter chỉ áp dụng được cho thẻ <p>.X
2bCó thể sử dụng thuộc tính font-size để thay đổi kích thước và thuộc tính color để thay đổi màu sắc của chữ cái đầu tiên.X
2cCó thể viết CSS như sau để định dạng chữ cái đầu tiên: p::first-letter { font-size: 200%; color: red; }.X
2dSử dụng phần tử giả ::first-letter để định dạng chữ cái đầu tiên của phần tử.X
3aCông ty X có thể viết CSS như sau: a:link { color: green; } a:visited { color: blue; }.X
3bCác liên kết có thể thay đổi màu sắc khi được hover chuột vào.X
3cCông ty X có thể sử dụng bộ chọn a:link và a:visited để định dạng màu sắc cho các trạng thái khác nhau của liên kết.X
3dCông ty X cần sử dụng thuộc tính color để thay đổi màu sắc của liên kết.X
4aBình có thể viết CSS như sau: h1 { font-size: 3em; font-weight: bold; color: darkblue; }.X
4bBình cần sử dụng thuộc tính font-size để thay đổi kích thước chữ.X
4cBình có thể sử dụng giá trị bold cho thuộc tính font-weight và giá trị darkblue cho thuộc tính color.X
4dBình cần sử dụng thuộc tính font-weight để thay đổi độ đậm của chữ.X
5aKhi hai bộ chọn có cùng mức ưu tiên, thuộc tính được khai báo sau cùng trong CSS sẽ được áp dụng.X
5bBộ chọn ID (ví dụ: #header) có mức độ ưu tiên cao hơn bộ chọn lớp (víệu: .header).X
5cCác bộ chọn kiểu (ví dụ: p, h1) có mức ưu tiên cao hơn bộ chọn ID.X
5dThuộc tính được áp dụng bằng cách sử dụng từ khóa !important sẽ luôn có mức độ ưu tiên cao nhất.X
6aMinh có thể sử dụng thuộc tính color để thay đổi màu sắc của liên kết.X
6bMinh có thể viết CSS như sau: a:link { color: blue; } a:visited { color: purple; }.X
6cMinh có thể sử dụng lớp giả :link cho trạng thái chưa truy cập và :visited cho trạng thái đã truy cập.X
6dMinh cần sử dụng lớp giả (pseudo-class) để định dạng các trạng thái của liên kết.X
7aCô giáo có thể sử dụng giá trị red cho thuộc tính border-color, giá trị 5px cho thuộc tính border-width và giá trị dashed cho thuộc tính border-style.X
7bCô giáo cần sử dụng thuộc tính border-color để thay đổi màu sắc của đường kẻ.X
7cCô giáo có thể viết CSS như sau: hr { border-color: red; border-width: 5px; border-style: dashed; }.X
7dCô giáo cần sử dụng thuộc tính border-width để thay đổi độ dày của đường kẻ.X
8aViệc kiểm tra phản hồi từ người dùng sau khi công bố trang web là rất quan trọng.X
8bThời gian tải trang không cần được xem xét khi kiểm tra chất lượng trang web.X
8cĐảm bảo rằng tất cả các liên kết trên trang web đều hoạt động đúng cách.X
8dKiểm tra tính tương thích của trang web với các trình duyệt khác nhau là không cần thiết.X
